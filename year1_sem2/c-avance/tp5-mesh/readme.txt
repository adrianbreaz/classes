
Sujet de l'étude:
1/ Ecrire une structure de données permettant de stocker un maillage 2D de cellules triangulaires
2/ Lire les données du maillage à partir de fichier de données du logiciel TRIANGLE
3/ Ecrire un maillage au format de sortie VTK.


Liens:
* logiciel TRIANGLE: http://people.sc.fsu.edu/~jburkardt/c_src/triangle/triangle.html
* logiciel Paraview (visualisation de fichier vtk): http://www.paraview.org/


Contenu du répertoire:

ce_répertoire
├── 4cell_mesh.vtk             : exemple de fichier VTK
├── data000.vtk                : exemple de fichier VTK
├── formatFichierTriangle.txt  : description du format de fichier du logiciel TRIANGLE
├── formatFichierVTK.txt       : description du format de fichier *.vtk
└── readme.txt                 : ce fichier.
